# Pierre-feuille-ciseau

## 🗒️ Guide

### Prérequis

- Copilot
- Copilot Chat
- Java 11+

---

Nous allons développer un mini jeu de pierre-feuille-ciseau très simpliste, qui va utiliser le terminal pour demander un des signes à l'utilisateur et en fonction d'un signe aléatoire, répondre victoire ou défaite de l'utilisateur

1) Ouvrez le fichier Lab-4\src\Main.java
2) Dans le fichier, 
    - Si vous avez l'auto-complete d'activé, écrivez un commentaire en java ```// write a rock paper scissors game```
    - Si ce n'est pas activé, écrivez le commentaire dans l'inline code (ctrl+I sur VSCode, ctrl+shift+G sur Intellij)
3) Appuyez sur entrée et laisser Copilot vous suggérer du code, si cela vous semble correct, validez avec Tab, ou bien essayez de modifier la description
4) Testez votre code en tapant la commande ```java .\Main.java``` avec un terminal localisé dans le ```Lab-4\src```.

(Si vous voulez un exemple de résultat attendu, vous pouvez regarder le fichier ```.\Main-exemple.java```)

5) Tentez d'ajouter d'autres éléments autour de ce code

ex: Nouvelle feature, Tests, PR, etc
