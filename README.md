### Comment ca marche

Tous les labs sont contenu dans le dossier "Instructions", chacun possède un fichier markdown d'instruction pour permettant de suivre les différents énoncés pour vous diriger sur l'utilisation de Github Copilot

Tous les exercices sont réalisable dans tous les modes de Copilot (Ask, edit, plan, agent), mais nous vous conseillons de les réaliser dans le mode Ask (sauf si l'énoncé vous demande de tester dans un mode en particulier)

Si vous êtes prêt, nous pouvons commencer le [Lab 1](Instructions/Labs/Lab-1-Github-Copilot-Modes.md)
